# Project_MovieInfo
A movie-info site which uses the api of OMDb to gather the infomation about the movie request.
The first page is a normal entry page <start.html>; this is a simple entry page with some visual aids.
Now the second page is the homepage of the site <main.html>; as this is a homepage, it contains some modified content.
The final page is the <searchPP.html>; this is the page where the use of api is done.
